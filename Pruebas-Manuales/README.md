# 🐞 Documentación de Pruebas Manuales & Aseguramiento de Calidad

En este directorio se consolidan los artefactos y evidencias derivados de la estrategia, diseño y ejecución de pruebas funcionales y no funcionales (caja negra), así como la gestión integral del ciclo de vida de los defectos en entornos Web y API.

---

## 📂 Contenido de la Sección

* **[Reporte de Defectos (Bug Report)](./Reporte_Bugs_Registro.md):** Documentación técnica detallada de fallos encontrados, incluyendo severidad, prioridad, pasos de reproducción y datos de entrada.
* **Gestión de Pruebas con Zephyr:** Estructuración de ciclos de ejecución, diseño de matrices de casos de prueba de UI y trazabilidad de requerimientos directamente integrados en Jira Software.
* **Validación de Servicios con Postman:** Integración de pruebas funcionales en la capa de Backend (APIs) para verificar la consistencia, lógica de negocio y códigos de estado HTTP de forma manual antes de la persistencia de datos.
